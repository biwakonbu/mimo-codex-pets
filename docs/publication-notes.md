# Publication Notes

This repository contains the public-safe Mimo pet package and QA assets.

Excluded local-only artifacts:

- live Codex screen recordings
- live recording thumbnails/contact sheets
- local rebuild working directories
- full generated image source directories under `~/.codex/generated_images`

Those files can reveal local desktop, browser, or workspace context and should not be committed to a public repository.

Curated state-specific raw strips are intentionally preserved under `sources/mimo/generated-raw/` because they are public-safe pet assets and are required for repeatable regeneration.
